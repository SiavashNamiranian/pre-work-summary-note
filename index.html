HTML (Hypertext Markup Language)

  • HTML provides the frame, divides, and defines webpage content to browsers and developers
  • Meta-related, content division and semantic elements define and divide a page
  • The head (metadata) element contains browser related information (accessibility, search engines and performance) about the webpage
  • The head elements are invisible for users, except title which appears on browser tab and is extremely important for search engine optimization
  • !DOCTYPE html is the declaration to browser that webpage utilizes HTML and is not case sensitive
  • The style element define CSS style of the page
  • The base element defines base URL for the page
  • The link element conects external resource to the HTML document
  • The meta element defines metadata (i.e. character set, description, keywords, author, and viewport)
  • The script element embeds client-side scripts in the HTML document
  • The body element represents the visible content shown to the user
  • h1 to h6 are headings in order, specially useful when screen readers used to access the page, h1 should only be used once
  • The header element in the body appears at top of the page and contains either navigation bar or introductory content
  • The img element contains source (src) of image and (alt) which is text describing appearance and functionality of the image (screen reader & search engine rankings imprv)
  • The section element allows webpage thematic content be organized by topic
  • The class/id attributes are used by HTML to identify and select specific element

CSS (Cascading Style Sheets)

  • Inline CSS are used to format individual element directly in HTML (i.e. h1 style="color:specify;" can assign color to h1 only)
  • The internal style sheet are CSS rules that are added in the head section using style element and only apply to that page
  • The external style sheet is a separate sheet/file containimg elements' formatting specifications
  • the Declaration block comes after selector and is composed of individual declaration {property: value;}
  • The link element (after last meta, before title) in the head allows external CSS (style.css) to be associated with HTML
  • The rel attribute defines relationship while href specifies the URL of external resource
  • A margin indicates how much space we want around the outside of the an element.
  • A padding indicates how much space we want around the content inside an element.
  • A . card {specifications} allows HTML elements with class card (ID) to be exclusively formated in the CSS
  • The hex # is used to specify the wider color spectrum as an alternative to just typing a color

Git

  • git clone: clones repo in local directory
  • git status: checks what branch we are currently on
  • git checkout main: switches to main branch
  • git checkout -b branch-name: creats a new branch and switches to it
  • git add -A: adds All modifications to the staging area_prepares git for commit
  • git commit -m "description of feature being added": snapshot of staged changes is created
  • git pull origin main: pull down most up to date version of main branch from Github, to which changes can be added
  • git push origin feature/descriptn: pushes changes up to github
